## 📱 **Caractéristiques du Design**

### **Typographie**
- **Police** : System fonts (-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto)
- **Tailles** : 0.875rem (14px) pour les textes, 1.875rem (30px) pour les titres
- **Poids** : 500-600 pour les éléments importants

### **Espacement**
- **Padding** : 0.75rem (12px) pour les inputs, 3rem (48px) pour les containers
- **Margins** : 1.5rem (24px) entre les éléments
- **Bordures** : 6px pour les éléments, 8px pour les containers

### **Animations**
- **Transitions** : 0.15s ease (plus rapides et subtiles)
- **Hover** : Changements de couleur discrets
- **Focus** : Bordures et ombres subtiles
