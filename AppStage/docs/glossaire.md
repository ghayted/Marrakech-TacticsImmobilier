## Glossaire

API REST Interface d’API web basée sur HTTP et JSON
ASP.NET Core Framework backend .NET de l’API
EF Core ORM .NET pour accéder à SQL Server
SQL Server Base de données relationnelle du projet
React Bibliothèque JavaScript pour le frontend
JWT JSON Web Token pour l’authentification
DTO Data Transfer Object pour les échanges API
Stripe Prestataire de paiement en ligne
PaymentIntent Objet Stripe représentant l’intention de paiement
Webhook (Stripe) Notification Stripe vers l’API
Facture PDF (iTextSharp) Document PDF généré après paiement
BiensImmobilier Entité représentant un bien immobilier
Reservation Entité représentant une réservation d’un bien
Disponibilite Calendrier/état de disponibilité et prix par nuit
Paiement Enregistrement d’un paiement lié à une réservation
